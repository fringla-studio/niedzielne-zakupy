# Support

Thanks for using **Niedzielne Zakupy**.

This repository is the best place to report public issues, suggest improvements, or share feedback about the app.

## Before you report an issue

Please check:

1. whether the app is updated to the latest version,
2. whether a similar issue already exists in GitHub Issues,
3. whether the problem happens again after restarting the app or refreshing widgets.

## What to include

For bugs, please include:

- app version,
- iOS or Android version,
- device model,
- app language,
- short description of the problem,
- steps to reproduce it,
- screenshot or screen recording, if helpful.

## Private matters

Do not post private information in public issues.

Avoid sharing:

- personal data,
- email addresses,
- payment details,
- full device identifiers,
- screenshots containing private notifications or personal information.

If private contact becomes necessary, a Fringla Studio contact address will be added here.
