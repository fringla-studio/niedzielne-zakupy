## What the app does

Niedzielne Zakupy helps answer one everyday question:

> **Can I shop today in Poland?**

The app shows:

- whether today is a trading or non-trading day,
- upcoming trading Sundays,
- selected public holidays with trading restrictions,
- a calendar and list view for upcoming dates,
- optional reminders,
- home-screen widgets for a quick glance.

The app is designed primarily for people living in Poland, but it can also help visitors, expats, and anyone who does not closely follow Polish trading Sunday rules.

## About this repository

This is the public GitHub repository for **feedback, issue tracking, and public product notes** for Niedzielne Zakupy.

**This is not the app source-code repository.**

You can use this repository to:

- report bugs,
- suggest improvements,
- request translation fixes,
- describe confusing screens or messages,
- share widget or notification problems,
- report accessibility problems,
- follow public product updates.

## Report a problem

Before opening a new issue, please check whether a similar issue already exists in Issues.

When reporting a bug, include:

- app version,
- platform: iOS or Android,
- device model,
- system version,
- language used in the app,
- what you tried to do,
- what you expected to happen,
- what happened instead,
- screenshots or screen recordings, if helpful.

Open a new issue here:

**[Report a bug](../../issues/new?template=bug_report.yml)**

## Suggest an improvement

Small, practical suggestions are the most useful.

Good examples:

- "The widget label is hard to read on dark wallpaper."
- "The text on this screen sounds too formal."
- "The calendar should make the next trading Sunday easier to spot."
- "I would like a reminder one day before a trading Sunday."

Open a new issue here:

**[Suggest an improvement](../../issues/new?template=feature_request.yml)**

## Translations

Niedzielne Zakupy is designed primarily for Polish users, but the app should also be clear for people using it in English or other languages.
