# Contributing

This is a public issue repository for **Niedzielne Zakupy**.

The app source code is not maintained here, but feedback is welcome.

## Useful contributions

You can help by:

- reporting bugs,
- describing confusing screens,
- suggesting small product improvements,
- reporting translation issues,
- sharing widget and notification problems,
- pointing out accessibility problems,
- adding context for device-specific issues.

## Good issue reports

A good issue is clear, specific, and easy to verify.

Please include:

- what you tried to do,
- what happened,
- what you expected,
- where it happened in the app,
- device and system version,
- app version,
- screenshots, when useful.

## Tone

Please keep issues practical and respectful.

Short, direct reports are perfect. There is no need to write long formal tickets.
